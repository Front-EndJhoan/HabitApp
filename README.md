# 📊 Habit Tracker App

## 🧠 Descripción

**HabitApp** es una aplicación diseñada para ayudarte a construir, mantener y mejorar tus hábitos diarios de forma sencilla e intuitiva. Su objetivo es ofrecer una experiencia clara y motivadora que permita a los usuarios visualizar su progreso y mantenerse constantes en el tiempo.

## 🚀 Características principales

* ✅ Seguimiento diario de hábitos
* 📈 Visualización del progreso
* 🎯 Objetivos y metas configurables
* 📅 Historial de actividad
* 🌙 Interfaz limpia y fácil de usar

## 🛠️ Tecnologías utilizadas

Esta aplicación ha sido desarrollada utilizando tecnologías modernas para garantizar rendimiento y escalabilidad:

* Frontend: [React / Typescript]
* Backend: [Node.js /]
* Base de datos: [MongoDB /]

## 📦 Instalación

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

```bash
# Clonar el repositorio
git clone https://github.com/Front-EndJhoan/HabitApp.git

# Acceder al directorio
cd HabitApp

# Instalar dependencias
npm install

# Ejecutar la aplicación
npm start
```

## 📌 Uso

1. Crea tus hábitos personalizados
2. Registra tu progreso diariamente
3. Visualiza estadísticas y tendencias
4. Ajusta tus objetivos según tus necesidades
